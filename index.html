Modo de Usar

  • Utilize um número de versão diferente da atual no dispositivo, independentemente de ser maior ou menor.
  • Guarde a API Key para futuras atualizações.
  • Gere uma nova API Key se ainda não tiver uma. Caso já tenha gerado, utilize a mesma, pois o código fará referência a ela.
  • Se desejar trocar a API Key, gere uma nova e insira-a na versão atualizada do código. No entanto, use a antiga para fazer o upload do novo firmware e, a partir da próxima atualização, utilize a nova.
  • Após enviar um firmware novo, o antigo será apagado. Portanto, faça backup do firmware anterior, caso precise reverter para uma versão antiga.
  • Sempre gere o binário do firmware com a partição adequada para OTA (Over-the-Air), caso contrário, a próxima atualização não será possível.
